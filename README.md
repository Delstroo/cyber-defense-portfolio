# Cyber Defense Portfolio

A responsive portfolio showcasing cybersecurity projects and technical skills, built with modern web technologies.

## 🚀 Tech Stack

- Vite + React + TypeScript  
- Tailwind CSS  
- shadcn/ui  

## 🛠 Getting Started

```bash
git clone https://github.com/Delstroo/cyber-defense-portfolio
cd cyber-defense-portfolio
npm install
npm run dev
```
## 🧪 Features
Clean UI with dark mode

Project cards with category filters

Responsive and mobile-friendly

Easy to customize and extend
